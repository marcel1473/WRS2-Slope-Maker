Winter Resort Simulator 2 - Slope Adder

How to use:
1. Run WinterSlopeTool.exe.
2. Click Browse and pick your savegame .lua file.
   Usual folder:
   Documents\My Games\WinterResortSimulator\savegames
3. Fill in:
   - Slope number/id
   - Info Layer number
   - Run name
   - From
   - To
   - Difficulty
   - Length seconds
   - Max skier capacity
4. Click Add Slope To File.
5. Start/load the save in the game.

The app creates a backup next to your save file before editing it.
The backup name looks like:
yourSave.lua.backup-20260503-153000

To rebuild the EXE:
Run build.bat from this folder.
